### Export .sql file in Database
- Create an folder for store .sql file
    - Login to the Database find out the database
        `SHOW DATABASES;`
    - backup command
    - `mysqldump -u your-username -p your-database-name > /home/ec2-user/backups/your_database_backup.sql`
- Example :
    `mysqldump -u your-username -p your-database-name > /home/ec2-user/backups/your_database_backup.sql`
